# 📦 Кейсы: Живой опыт и Смыслы

В этой папке хранятся черновики и разборы конкретных съемок. 
Даша, просто накидывай сюда факты, а я буду вытаскивать из них "мясо" для постов.

## Шаблон кейса:
- **Кто/Где:** (Например: Эксперт по йоге, скалы Нячанга)
- **Запрос:** (Что хотел клиент?)
- **Сложность:** (Что пошло не так или в чем был вызов?)
- **Решение:** (Как сработала режиссура кадра и стилизация?)
- **Результат:** (Какие смыслы удалось подсветить?)
